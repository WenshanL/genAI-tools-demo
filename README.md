## 🔧 工具類型

### 💡 [應用工具](tools/application.md) 
針對特定應用場景所設計的工具，通常有具體的功能或專業解決方案，如文案生成、聲音轉換等，**使用者可直接使用不須透過程式加工即能達到應用目的**。

### ⚙️ [開發工具](tools/development.md) 
提供基礎架構或元件，用於開發、佈署、優化追蹤等，因為需要透過撰寫程式達到應用目的，**使用對象通常是程式開發者**。

### 🧠 [語言模型](tools/model.md)
主要是大型語言模型或用於自然語言處理相關的模型
